# coop_system

This repo is version 2 of coop system in DSSI UBU.
**ระบบติดตามกระบวนการสหกิจศึกษา**

กระบวนการฝึกสหกิจของนักศึกษา เริ่มจากนักศึกษาที่จะไปฝึกสหกิจได้ต้องผ่านการอบรมเตรียมความพร้อมเป็นจำนวน 30 ชั่วโมงขึ้นไป และอาจารย์อนุมัติให้ไปฝึกสหกิจกับบริษัทที่สมัครไป ระหว่างฝึกให้มีการรายงานผลการปฏิบัติสหกิจรายสัปดาห์จำนวน 15 สัปดาห์ หลังจากฝึกเสร็จสิ้นนั้นบริษัทจะทำการประเมินผลการทำงานว่งให้อาจารย์เพื่อเป็นคะแนนต่อไป

นักศึกษาสามารถสมัครเข้าใช้งาน และลอกอินเข้ามาในระบบได้ ดูความก้าวหน้าในกระบวนการฝึกสหกิจของตนเองได้ สามารถอ่านข่าวหรือประกาศที่อาจารย์แจ้งมาได้สามารถอัพเดทข้อมูลการอบรมเตรียมความพร้อม โดยให้ระบุชื่อหัวข้อที่อบรมและจำนวนชั่วโมงที่อบรม และเมื่อจำนวนชั่วโมงฝึกอบรมครบ 30 ชั่วโมงขึ้นไป นักศึกษาจะสามารถสมัครงานสหกิจศึกษาได้ โดยให้บบันทึกข้อมูลบริษัท ตำแหน่งหน้าที่ ช่วงวันที่ไป และชื่อผู้ดูแลงาน  และเมื่องานที่สมัครได้รับอนุมัติและอยู่ในช่วงฝึกงานที่ระบุก็สามารถบันทึกข้อมูลรายงานผลการปฏิบัติงานรายสัปดาห์ได้

ตัวแทนบริษัทสามารถลอกอินมาประเมินผลการฝึกสหกิจของนักศึกษาที่อยู่ในบริษัทได้

อาจารย์สามารถลอกอินเข้ามาเพื่อ ประกาศแจ้งข่าวนักศึกษารายคนหรือทั้งระบบได้ อัปโหลดเอกสารให้นักศึกาสามารถดาวน์โหลดได้ สามารถตรวจสอบข้อมูลการฝึกอบรมของนักศึกษาโดยกดยินยอมพร้อมระบุชั่วโมงที่ได้รับหรือปฏิเสธได้ สามารถตรวจสอบข้อมูลการสมัครสหกิจโดยกดยินยอมหรือปฏิเสธได้ สามารถดูรายงานผลการปฏิบัติงานรายสัปดาห์พร้อมกดเครื่องหมายตรวจสอบแล้วได้ สามารถสร้างแอคเคาท์ให้ตัวแทนบริษัทเข้ามาประเมินได้ สามารถดูผลการประเมินของนักศึกษารายคนได้ และสามารถดูความก้าวหน้าในกระบวนการฝึกสหกิจของนักศึกษาแต่ละคนได้
